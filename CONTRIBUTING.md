# Contributing

Thank you for your interest in contributing to this project.

The canonical contributing guide for all Phenotype-org repositories lives at:

https://github.com/KooshaPari/Phenotype/blob/main/CONTRIBUTING.md

Please read that document before opening issues, pull requests, or discussions
specific to this repository. It covers:

- Code of conduct
- Development setup
- Coding standards and quality gates
- Pull request and review process
- Issue triage
- Release and versioning policy

Issues and PRs in this repository should follow the conventions described there.
